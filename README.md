# enumerable-methods

![](https://img.shields.io/badge/Microverse-blueviolet)

In this project we implemented Ruby's Enumerable methods.

# Enumerable Ruby Methods

> There we have:

- my_each
- my_each_with_index
- my_select
- my_all?
- my_any?
- my_none?
- my_count
- my_map
- my_inject
- multiply_els

## Built With

- Ruby

## Tested with Rspec

In order to test the methods run the command in terminal:

`rspec`
